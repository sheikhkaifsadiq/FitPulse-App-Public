# FitPulse (User App UI Showcase)

Welcome to the public UI showcase for FitPulse. This repository contains the frontend UI components, screens, layouts, and public assets developed for Phase 1 (User Side) of the mobile application.

The app is actively under development. Phase 1 (User App) UI is complete, and Phase 2 (Expert Portal) is currently in the roadmap.

🔒 **Security & Architecture Note**
This public repository serves as a demonstration of the mobile application\'s UI/UX, architecture, and tech stack. For security and proprietary reasons, sensitive business logic, database integrations, and proprietary algorithms are maintained in a separate private repository. Only safe, structural frontend code is showcased here.

⚠️ **Compilation Notice**
Note: Because all proprietary ViewModels, data layers, and backend integrations have been intentionally removed for security purposes, this repository serves as a read-only architectural showcase and will not compile locally.
